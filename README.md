## 🛠️ Customization

To personalize your portfolio, edit the following files:

### `index.html`
Update the content sections:
- Change achievement descriptions and metrics
- Add your contact email
- Modify the introductory text
- Update any other personal information

### `styles.css`
Customize the appearance:
- Change the color scheme by updating CSS variables in `:root`
- Modify fonts, spacing, and layout
- Adjust responsive breakpoints
