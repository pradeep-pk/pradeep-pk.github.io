## How to Customize

### Edit Your Information
- Open `index.html`
- Update the skills, projects, and about section with your details
- Add your actual projects and descriptions

### Change Colors
- Open `styles.css`
- Update the CSS variables in the `:root` section
- Current theme uses blue (#2563eb)
